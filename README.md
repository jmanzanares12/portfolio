Mi portfolio personal diseñado para mostrar mi trayectoria como Ingeniero de Software. Una aplicación moderna construida completamente con **React** y **Vite**, enfocada en la interactividad, el rendimiento y una estética profesional.

## Objetivo del Proyecto

Este sitio sirve como mi centro de operaciones digital, donde demuestro mi capacidad para transformar lógica de negocio compleja (ERP, Sistemas) en interfaces de usuario fluidas y atractivas.

## Características Técnicas

- **Vite Powered:** Desarrollo y compilación ultra rápida para una experiencia de usuario sin fricciones.
- **Component-Based Architecture:** Estructura modular y escalable utilizando hooks personalizados y componentes reutilizables.
- **Premium UI:** Implementación de modo oscuro nativo, animaciones sutiles y maquetación con Tailwind CSS.
- **Adaptive Design:** Optimización total para dispositivos móviles y pantallas de alta resolución.
- **Smooth Navigation:** Transiciones fluidas entre secciones para una navegación inmersiva.

## Stack Tecnológico

- **Library:** [React.js](https://reactjs.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Logic:** [TypeScript](https://www.typescriptlang.org/)
- **Deployment:** [Vercel](https://vercel.com/)
